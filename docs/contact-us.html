Contact us

You can contact us using the below methods:

  • streetmanager@dft.gov.uk - for general enquiries to the Street Manager project
  • register@manage-roadworks.service.gov.uk - to request access to Street Manager and for any questions related to activation of access
  • betasupport@streetmanager.atlassian.net - to raise tickets for any technical issues with Street Manager
  • #general channel on DfT Street Manager Slack - for general enquiries to the Street Manager project
  • #askthedft channel on DfT Street Manager Slack - for any questions to DfT Street Manager
  • #businesschange channel on DfT Street Manager Slack - for any business change related questions to DfT Street Manager
