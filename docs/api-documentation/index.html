API documentation

Note that this documentation is being trialled for use with the Street Manager Beta and is aimed towards authorised Beta participants. As Street Manager remains under active development, this documentation will continue to develop over time.

Overview

This page contains guidance for developers integrating their systems with the Street Manager API so they can submit and retrieve information about street works.

The page is split into two sections. The first contains specification documentation which describes the legally required API functions and interfaces which are due to be delivered in the coming weeks and months. The second section lists the versioned and dated releases, as the specified functionality is implemented and deployed into the service environments.

Specification Documentation

Version 1 - Stable

  • Version 1.12

This is the V1 baseline specification which describes all legally required API functions and interfaces. This specification includes any breaking changes planned to be introduced in upcoming releases; while new functionality may be introduced in future versions of the V1 specification, these changes will be purely additive in nature to prevent existing integrations from breaking.
